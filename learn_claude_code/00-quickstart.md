# 0.5 Quickstart（最短で体験）

この章は「最短で Claude Code を体験する」ための流れです。
5分で 1 回成功体験を作ることを目的にしています。

## 0.5.1 最小プロンプト
```
目的: README に記載されたセットアップ手順を要約して
制約: 変更はしない、読み取りのみ
完了条件: 5 行以内で要約される
```

## 0.5.1.1 実行サンプル（もう1本）
```
目的: app/api/README.md にある API 一覧を表形式で整理して
制約: 変更はしない、読み取りのみ
完了条件: 5 行以内の表でまとめる
```

## 0.5.2 失敗した時の修正ループ
1. うまくいかなかった点を 1 つ書く
2. その点を制約 or 完了条件に追加する
3. もう一度実行する

## 0.5.3 良い / 悪いの例（最小プロンプト）

| 観点 | 悪い例 | 良い例 |
| --- | --- | --- |
| 目的 | いい感じに調べて | README を要約して |
| 制約 | 指定なし | 変更はしない、読み取りのみ |
| 完了条件 | 指定なし | 5 行以内で要約 |

## 0.5.4 演習テンプレ
目的: 最小プロンプトを 1 本作る
入力: README の内容（読める範囲）
出力:
```
目的:
制約:
完了条件:
```

### 解答例
```
目的: README に書かれたセットアップを要約する
制約: 変更はしない
完了条件: 5 行以内
```
